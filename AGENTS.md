# Intelligence Brief Agent

## Agents
- `orchestrator`: Controlador principal del pipeline
- `product-owner`: Descompone queries en dimensiones
- `architect`: Diseña el grafo LangGraph
- `developer`: Implementa el pipeline en código
- `devops`: Gestiona infraestructura y secrets
- `qa`: Valida factualidad y calidad

## Skills
- `tavily-search`: Búsqueda web con Tavily API
- `qdrant-query`: Búsquedas vectoriales en Qdrant
- `brief-structure`: Template del reporte final
- `loop-engineering`: Loop de mejora iterativa

## Commands
- `python scripts/main.py`: Ejecutar pipeline
