# Definition of Done (DoD)

## Criterios de Calidad del Reporte

### Executive Summary
- [ ] Máximo 300 palabras
- [ ] Cubre hallazgos principales
- [ ] Incluye recomendación clave
- [ ] Sin jerga técnica innecesaria

### Análisis por Dimensión
- [ ] Mínimo 3 fuentes web por dimensión
- [ ] URLs verificadas y accesibles
- [ ] Fechas de publicación claras
- [ ] Separación explícita: hechos vs interpretación
- [ ] Confianza estimada (alta/media/baja)

### Contexto Histórico
- [ ] Referencia a datos previos en Qdrant
- [ ] Coherencia con noticias recientes
- [ ] Identificación de tendencias

### Proyecciones
- [ ] Basadas en evidencia factual
- [ ] Múltiples escenarios considerados
- [ ] Probabilidades estimadas
- [ ] Supuestos explícitos

### Metodología
- [ ] Número total de fuentes consultadas
- [ ] Período cubierto documentado
- [ ] Profundidad de búsqueda indicada
- [ ] Iteraciones de mejora registradas

### Fuentes
- [ ] Tabla completa de fuentes
- [ ] URLs funcionales
- [ ] Fechas de acceso
- [ ] Relevancia scored
